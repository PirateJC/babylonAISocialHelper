# BabylonSocialHelper Review Web App - Mock Context

## Original Description

A static web application hosted on GitHub Pages that lets the Babylon.js core team review, edit, and approve AI-generated social media posts. The app uses GitHub OAuth for authentication (with BabylonJS team membership check), imports posts from a JSON file, auto-assigns dates, and commits approved posts to the repo's `scheduled/` directory for a GitHub Actions workflow to publish.

Key screens:
- Login (GitHub OAuth)
- Post list view with filtering, batch select, batch approve/delete
- Individual post view with generic preview, editing, and approval
- JSON import flow with auto-date assignment
- Failed post view showing per-platform error details
